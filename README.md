# Foresight  Frontend

Frontend for the Foresight prediction market platform. Built with vanilla React and deployed on Netlify. Connects to GenLayer Bradbury testnet.

Live at https://foresightmrkts.netlify.app

## Network

| Parameter | Value |
|-----------|-------|
| Network | GenLayer Bradbury Testnet |
| Chain ID | 4221 |
| RPC | https://rpc-bradbury.genlayer.com |
| Explorer | https://explorer-bradbury.genlayer.com |
